## Данные
- год продажи (2019);
- месяц продажи (январь - сентябрь);
- название автоцентра;
- название продаваемой марки автомобиля;
-  название модели автомобиля;
-  модификация модели автомобиля;
- год производства автомобиля;
- страна производителя;
- вид топлива;
- объём двиг л;
- коробка передач;
- тип привода;
- сегмент, к которому относится авто;
- регион продажи;
- наименование дилерского центра;
- тип клиента (юр. или физ. лицо);
- форма расчета;
- количество автомобилей в заказе;
- цена USD;
- цена заказа USD;
- область продажи;
- сегмент автомобиля актуальный;
- класс автомобиля актуальный;
- английская сегментация;
- kокализация производства.
### Задача
В датасете представлены данные о продажах автомобилей на территории Казахстана в январе-сентябре 2019 года всеми официальными автодилерами, которые собрала статистическая компания-агрегатор.

Данные содержат ошибки и в текущем виде не пригодны для анализа. Необходимо очистить данные до состояния, когда их можно анализировать и провести исследовательский анализ данных + анализ рынка автомобилей Казахстана.
