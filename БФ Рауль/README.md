# [Проект БФ Рауль дашборд](https://datalens.yandex/nal0kvwcxfw0c)
## Данные
- id кандидата
- пол
- возраст
- дата добавления кандидата
- наличие инвалидности
- наличие сиротского опыта
- наличие адаптированной образовательной программы
- дата начала трудоустройства
- дата завершения трудоустройства
- проработал ли кандидат более 1 месяца
- тип площадки
- тип трудоустройства
- тип опыта работы
- код региона
- код провайдера.
### Задача
Необходимо построить дашборд, отображающий динамику и структуру трудоустройств.
-	Параметры динамики: помесячно, поквартально, ежегодно.
-	Параметры структуры: принадлежность благополучателя к одному из сегментов, ответственная НКО, регион, пол благополучателя, возраст, тип трудоустройства (временное, постоянное с сопровождением, постоянное без сопровождения), тип работодателя (внутренняя защищённая площадка, внешняя защищённая площадка, открытый рынок).
- Параметры трудоустройства: от 1 дня, от 1 месяца. 

Также провести исследовательский анализ данных.
